Preguntas a responder

¿Qué hace la función phpinfo()? Describe y discute 3 datos que llamen tu atención.

La funcion phpinfo(), nos muestra información, sobre el sistema operativo que se esta utilizando, el servidor apache y version con la cual se esta trabajando y toda la información sobre PHP[1]

¿Qué cambios tendrías que hacer en la configuración del servidor para que pudiera ser apto en un ambiente de producción?

Me tendría que asegurar que la configuración permita el uso de otros elementos para poder realizar el desarrollo (como sql, por ejemplo).[2]

¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.

Esto se logra por medio de http requests, donde el archivo hace requests al servidor externo de tal manera que éste regresa código html para que este pueda ser ejecutado por el navegador.[3]
